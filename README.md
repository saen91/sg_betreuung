Dieses Plugin ist vor allem für das Storming Gates erstellt wurden. <br>
Es ist ein Plugin mit dem es für manche Gruppen (über Einstellungen einstellbar) möglich ist, Themen zu betreuen/beobachten. Die Anzeige mit "wird betreut von..." kann über die Einstellungen angepasst werden. <br>
Sollte ein User irgendwann nicht mehr in der Gruppe sein, welche Themen beobachten darf, wird es im ModCP extra in der Übersicht aufgelistet. <br>
Man hat dann die Möglichkeit den User einfach rauszunehmen oder das Thema selbst zu übernehmen. <br><br>

Folgende Variablen müssen noch eingegeben werden:<br>
showthread:<br>
{$caretaker_output} wenn man einen kompletten Block aus "betreuen/nicht betreuen" und dem Namen haben will. <br>
{$caretaker_name} Für die reine Usernamen ausgabe vom betreuten <br>
{$caretaker_button} zum hinzufügen von Betreuern oder auch dem abbestellen vom Betreuen. <br><br>

forumdisplay_thread<br>
{$thread['looked_output']}: für den Button in der Themenübersicht des Forums zum hinzufügen & auch Namen anzeigen. <br><br>
