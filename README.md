Dieses Plugin ist vor allem für das Storming Gates erstellt wurden. 
Es ist ein Plugin mit dem es für manche Gruppen (über Einstellungen einstellbar) möglich ist, Themen zu betreuen/beobachten. Die Anzeige mit "wird betreut von..." kann über die Einstellungen angepasst werden. 
Sollte ein User irgendwann nicht mehr in der Gruppe sein, welche Themen beobachten darf, wird es im ModCP extra in der Übersicht aufgelistet. 
Man hat dann die Möglichkeit den User einfach rauszunehmen oder das Thema selbst zu übernehmen. 

Folgende Variablen müssen noch eingegeben werden:
showthread:
{$caretaker_output} wenn man einen kompletten Block aus "betreuen/nicht betreuen" und dem Namen haben will. 
{$caretaker_name} Für die reine Usernamen ausgabe vom betreuten 
{$caretaker_button} zum hinzufügen von Betreuern oder auch dem abbestellen vom Betreuen. 

forumdisplay_thread
{$thread['looked_output']}: für den Button in der Themenübersicht des Forums zum hinzufügen & auch Namen anzeigen. 

Link zur Übersicht: 
https://deinforum.de/modcp.php?action=sg_betreuung

WAS NOCH FEHLT:
- Benachrichtigung wenn in dem Thema was neues gepostet wird
